## Django `.po` File Generation

From this table, generate `.po` files:

```
locale/en/LC_MESSAGES/django.po
locale/ru/LC_MESSAGES/django.po
locale/de/LC_MESSAGES/django.po
```

**Example entry:**

```
#: base.html:line
msgid "nav_home"
msgstr "Home"
```
