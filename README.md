# codex-portfolio

## Структура проекта

```
├── deploy/             # Конфигурации для деплоя (Docker, Nginx и т.д.)
├── docs/               # Та самая документация по вашему стандарту
│   ├── architecture/   # Логика сплит-экрана и Bento-сетки
│   ├── design/         # Цвета Python, шрифты, UI-кит
│   └── meta/           # Инструкции для меня (AI) по этому проекту
├── scripts/            # Утилитарные скрипты (генерация структуры и т.д.)
├── src/                # Код самого сайта
├── assets/             # Те самые стилизованные карточки (фоны)
└── README.md           # Главный файл проекта
```
